# Tab Bar Icons

需要 8 张 81×81 的 PNG 图标文件：

| 文件名 | 颜色 | 说明 |
|--------|------|------|
| `tab-home.png` | `#999999` | 首页 - 未选中 |
| `tab-home-active.png` | `#07C160` | 首页 - 选中 |
| `tab-cal.png` | `#999999` | 日历 - 未选中 |
| `tab-cal-active.png` | `#07C160` | 日历 - 选中 |
| `tab-stats.png` | `#999999` | 统计 - 未选中 |
| `tab-stats-active.png` | `#07C160` | 统计 - 选中 |
| `tab-me.png` | `#999999` | 我的 - 未选中 |
| `tab-me-active.png` | `#07C160` | 我的 - 选中 |

可通过以下方式获取图标：
- [iconfont.cn](https://www.iconfont.cn/) 下载后放入此目录
- 微信开发者工具内置图标库
- 设计师提供
