## 1. 安装工具

### 1.1 下载微信开发者工具

打开官网：**https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html**

选择 **Windows 64位** 版本下载安装。

安装完成后打开，用微信扫码登录。

---

## 2. 注册微信小程序账号

### 2.1 注册

打开 **https://mp.weixin.qq.com/** ，点击右上角"立即注册"，选择"小程序"。

按提示填写邮箱、密码等信息。邮箱需要是**未注册过微信公众号/小程序的**。

注册过程中需要：
- 填写主体信息（个人就选"个人"，需要身份证号和微信绑定银行卡的实名认证）
- 管理员微信扫码验证

### 2.2 获取 AppID

注册完成后登录小程序后台 **https://mp.weixin.qq.com/** ，左侧菜单：

> **开发** → **开发管理** → **开发设置**

复制页面上的 **AppID（小程序ID）**，类似 `wxabcd1234efgh5678`。

---

## 3. 开通云开发环境

### 3.1 打开项目

1. 启动**微信开发者工具**
2. 点击 **+** 号 → **导入项目**
3. 目录选择：`D:\桌面\nano_personal1\wechat-exerciseplan`
4. AppID 填你在上面复制的那个
5. 项目名称随便填，比如"健身打卡"
6. 点击"导入"

### 3.2 开通云开发

项目打开后：

1. 点击顶部工具栏左侧的 **云开发** 按钮（云朵图标）
2. 在弹出的面板中点击 **开通**
3. 环境名称随便填，比如 `fitness`
4. 点击确定，等待 1-2 分钟创建完成

### 3.3 获取环境 ID

云开发控制面板打开后，顶部会显示环境名称和 ID。

环境 ID 类似 `fitness-1a2b3c`。**复制这个环境 ID**。

---

## 4. 配置项目文件

在微信开发者工具左侧文件列表中，找到并编辑以下两个文件：

### 4.1 配置云环境 ID

打开 `miniprogram/env.js`，把 `your-env-id` 替换为你的环境 ID：

```javascript
// 改之前
const CLOUD_ENV_ID = 'your-env-id';

// 改之后（用你自己的环境ID）
const CLOUD_ENV_ID = 'fitness-1a2b3c';
```

保存（Ctrl+S）。

### 4.2 配置 AppID

打开 `project.config.json`，找到这一行：

```json
"appid": "your-appid-here",
```

替换为：

```json
"appid": "wxabcd1234efgh5678",
```

保存。然后点右上角 **详情** → **基本信息**，确认 AppID 已经变成你填的。

**重要：修改 project.config.json 后需要关闭项目，重新导入一次让配置生效。**

## 5. 导入种子数据

小程序现在有动作库和训练计划模板的种子数据，需要导入到云数据库。

### 5.1 创建集合

1. 点击顶部 **云开发** 按钮打开云开发控制台
2. 左侧点 **数据库**
3. 点击 **+ 添加集合**，依次创建以下 5 个集合：
   - `users`
   - `exercises`
   - `training_plans`
   - `checkins`
   - `subscriptions`

### 5.2 导入动作数据

1. 在数据库页面点击 `exercises` 集合
2. 点击 **导入** 按钮
3. 选择文件：`database/seed-exercises.json`
4. 冲突模式选 **Insert（总是插入）**
5. 点击确定

导入成功后会看到 40 条动作数据。

### 5.3 导入训练计划模板

1. 切换到 `training_plans` 集合
2. 点击 **导入**
3. 选择文件：`database/seed-plans.json`
4. 冲突模式选 **Insert（总是插入）**
5. 点击确定

导入成功后会看到 8 条计划模板。

### 5.4 创建索引（可选但推荐）

在云开发控制台每个集合的 **索引管理** 标签页中，按设计文档添加对应索引。跳过这步不影响开发测试。

## 6. 部署云函数

项目中的 6 个云函数需要上传到云端才能被小程序调用。

在微信开发者工具左侧文件列表中，找到 `cloudfunctions` 目录。对下面每个子目录做相同操作：

### 6.1 依次部署 6 个云函数

右键点击 `cloudfunctions/user` → **上传并部署：云端安装依赖**

等待上传完成（状态栏会有提示）。完成后该云函数图标会变成云朵样式。

对以下每一个**依次重复**：
- `cloudfunctions/user` → 上传并部署
- `cloudfunctions/exercise` → 上传并部署
- `cloudfunctions/plan` → 上传并部署
- `cloudfunctions/checkin` → 上传并部署
- `cloudfunctions/statistics` → 上传并部署
- `cloudfunctions/notification` → 上传并部署

> 提示：每次部署大约需要 30-60 秒，总共约 5 分钟。请确保网络正常。

### 6.2 验证部署

点击 **云开发** 按钮 → **云函数**，应该能看到 6 个函数都在列表中，状态为"部署成功"。

## 7. 获取 Tab 图标

小程序底部 Tab 栏需要 8 张图标（4 个图标 × 2 种状态）。你需要准备 **81×81 像素的 PNG** 图标。

### 方式一：用图标网站下载（推荐）

1. 打开 **https://www.iconfont.cn/** ，注册登录
2. 搜索以下图标（或用任意你喜欢的）：
   - 首页 / home
   - 日历 / calendar
   - 图表 / chart
   - 用户 / user
3. 鼠标悬停图标 → 点击购物车图标加入
4. 点击右上角购物车 → **下载代码**
5. 颜色选 `#999999`（未选中灰），下载 PNG，尺寸 81px
6. 同样步骤再下载一套，颜色选 `#07C160`（微信绿），作为选中态

### 方式二：用微信自带图标

在微信开发者工具中，可以用组件库自带的图标，但这需要代码调整。建议方式一。

### 放入项目

下载后将 8 张图片改名为：

```
miniprogram/images/
├── tab-home.png          （灰色首页）
├── tab-home-active.png   （绿色首页）
├── tab-cal.png           （灰色日历）
├── tab-cal-active.png    （绿色日历）
├── tab-stats.png         （灰色统计）
├── tab-stats-active.png  （绿色统计）
├── tab-me.png            （灰色我的）
├── tab-me-active.png     （绿色我的）
```

## 8. 注册订阅消息模板

训练提醒功能需要微信订阅消息。如果你暂时不需要提醒功能，**你可以跳过这一节**。

### 8.1 选用模板

1. 登录小程序后台 **https://mp.weixin.qq.com/**
2. 左侧菜单：**功能** → **订阅消息**
3. 如果没有开通，点击开通
4. 在公共模板库搜索"提醒"，选用一个合适的模板（如"训练提醒"或"日程提醒"）
5. 选用后获得一个 **模板 ID**，类似 `abc123def456ghi789jkl`

### 8.2 配置到代码

在以下两个文件中，把 `your-template-id` 替换为你的模板 ID：

1. `miniprogram/pages/profile/profile.js`：
   ```
   搜索: tmplIds: ['your-template-id']
   改为: tmplIds: ['abc123def456ghi789jkl']
   ```

2. `cloudfunctions/notification/index.js`：
   ```
   搜索: 'your-template-id'
   改为: 'abc123def456ghi789jkl'
   ```

注意：`profile.js` 里有两处，都要改。

修改后重新部署 `notification` 云函数（参考第 6 节）。

## 9. 测试运行

### 9.1 编译运行

在微信开发者工具中点击顶部 **编译** 按钮。如果一切正常，你会看到：

- 底部 4 个 Tab 页
- 首页显示"还没有训练计划"的空状态

### 9.2 测试完整流程

1. 点击"选择训练计划" → 进入推荐模板列表
2. 点击任意模板 → 查看详情
3. 点击"复制此计划" → 计划被复制到自己名下
4. 返回首页 → 应该能看到今日的训练动作
5. 点击"开始打卡" → 填写组数、次数、重量
6. 点击"完成打卡" → 打卡成功
7. 切换到"日历"Tab → 看到今日有打卡标记
8. 切换到"统计"Tab → 看到打卡数据统计
9. 切换到"我的"Tab → 查看个人信息和设置

### 9.3 真机预览

点击开发者工具顶部的 **预览** 按钮，用手机微信扫描二维码，可以在手机上体验真实效果。

## 10. 提交审核上线

当你在开发工具和真机上都测试没问题后：

1. 点击开发者工具顶部的 **上传** 按钮
2. 版本号填 `1.0.0`，描述填"健身打卡小程序首次发布"
3. 上传成功后，登录小程序后台 **https://mp.weixin.qq.com/**
4. 左侧菜单：**管理** → **版本管理**
5. 在开发版本列表中找到刚上传的版本，点击 **提交审核**
6. 填写审核信息（功能描述、测试账号等）
7. 提交后等待 1-7 个工作日审核
8. 审核通过后在版本管理中点击 **发布**

## 常见问题

**Q: 编译时提示 "未找到 xx 页面"**
A: 检查文件是否在正确的路径下，或者点击"编译"旁边的箭头 → "清除缓存" → "重新编译"。

**Q: 云函数调用报错 "env not found"**
A: 确认 `miniprogram/env.js` 中的环境 ID 填写正确，且云函数已部署。

**Q: Tab 图标不显示**
A: 确认图标文件已放入 `miniprogram/images/` 目录，且文件名和 `app.json` 中配置的一致。

**Q: 数据库没有数据**
A: 回到第 5 节，确认已导入 `seed-exercises.json` 和 `seed-plans.json`。

**Q: 小程序预览时白屏**
A: 点右上角"详情" → "本地设置" → 勾选"不校验合法域名"（仅开发阶段使用）。
